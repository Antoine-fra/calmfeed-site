Vos choix de confidentialité

Dernière mise à jour : 13 juillet 2026

Feed ne collecte aucune donnée personnelle : pas de compte Feed, pas de serveurs, pas de publicité, pas de tracking, aucun SDK tiers. Il n'existe donc aucune collecte à laquelle s'opposer, aucune donnée à vendre ou partager, et aucun profil publicitaire vous concernant.

Refuser un suivi publicitaire

Sans objet : Feed n'effectue aucun suivi et n'affiche aucune publicité. Vous n'avez rien à désactiver.

Supprimer les données stockées sur votre appareil

Les seules données de Feed (préférences, réponses de l'onboarding, session web chiffrée) vivent dans le bac-à-sable de l'application, sur votre iPhone uniquement. Pour les supprimer :

  • Désinstallez l'application : toutes ses données locales sont effacées immédiatement et définitivement par iOS.

Supprimer le compte du site web tiers

Le compte que vous utilisez dans Feed appartient au site web que vous consultez, pas à Feed. Pour le supprimer : dans l'application, ouvrez Réglages → Compte → Supprimer le compte — vous serez dirigé vers la page officielle de suppression du site, où la suppression se termine selon ses conditions.

Abonnement

L'abonnement Feed est traité par Apple. Nous ne voyons ni votre identité, ni votre moyen de paiement. Gestion et annulation : Réglages iOS → votre nom → Abonnements, ou apps.apple.com/account/subscriptions.

Exercer un droit ou poser une question

Pour toute demande liée à la confidentialité : support@feedr.app. Voir aussi notre politique de confidentialité complète.
