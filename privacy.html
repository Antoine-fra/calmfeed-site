Politique de confidentialité

Dernière mise à jour : 8 juin 2026

Feed est conçu pour respecter votre vie privée. Cette politique explique précisément ce que l'application fait et ne fait pas avec vos données.

Données que Feed collecte

Aucune. Feed ne collecte, ne stocke, ni ne transmet aucune donnée personnelle vers nos serveurs ou des serveurs tiers. Nous n'avons pas de serveurs.

Données stockées localement sur votre appareil

Feed stocke uniquement les éléments suivants dans le bac-à-sable de l'application sur votre iPhone, jamais transmis ailleurs :

  • Vos préférences (verrou Face ID, filtre niveaux de gris) et les réponses de l'onboarding (estimation d'usage, âge, objectif)
  • Le statut de complétion de l'onboarding
  • La session de navigation web (cookies du site visité), chiffrée par iOS Data Protection

Tous ces éléments sont automatiquement supprimés si vous désinstallez l'application.

Services web tiers

Feed inclut une vue web qui charge un site externe choisi par l'utilisateur. Lorsque vous utilisez ce service via Feed :

  • Le site visité peut collecter des données vous concernant selon sa propre politique de confidentialité
  • Feed n'intercepte pas, ne lit pas, et ne transmet pas ces données
  • La collecte par le site visité a lieu exactement comme si vous utilisiez le navigateur Safari d'Apple
  • Feed n'envoie aucune donnée vers des tiers au-delà de ce que le site capte naturellement via votre navigation

Permissions iOS demandées

  • Caméra : utilisée uniquement quand vous tapez vous-même un bouton de capture dans le site web chargé. Aucun accès silencieux.
  • Bibliothèque photo : utilisée uniquement quand vous choisissez vous-même une photo à partager.
  • Microphone : utilisé uniquement quand vous enregistrez vous-même un message vocal dans le site web chargé.
  • Face ID : utilisé uniquement pour le verrouillage local de l'application.

Feed ne déclare aucun mode d'arrière-plan et n'envoie aucune notification : l'application ne s'exécute pas en arrière-plan. Aucune permission n'est utilisée à des fins de profilage, de publicité, ou de tracking.

Suivi publicitaire et analytics

Aucun. Feed n'embarque aucun SDK de tracking, d'analytics, de publicité, ou de crash reporting. Il n'y a strictement aucune télémétrie.

Vos droits (RGPD)

Comme Feed ne collecte aucune donnée, il n'y a rien à demander, modifier ou supprimer auprès de nous. Pour les données collectées par les sites web tiers que vous visitez via l'application, adressez-vous directement à ces services.

Modifications

Toute modification de cette politique sera publiée sur cette page. La date de dernière mise à jour est indiquée en haut.
